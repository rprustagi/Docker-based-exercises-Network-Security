Introduction

This course will provide an introduction to Network Security, which involves many understanding of networks, protocols and technologies that are woven together and their relationship to security. This course will help students understand how various attacks work and how to defend against them. The course follows the hands-on approach i.e. learn by doing.

Overview

This course will begin with a broad overview of Security concepts and threats and review of Network Protocols, network diagnostic tools, and countermeasures at each layer of Network Stack i.e. Application, Transport, Network and Link Layer.

Confidentiality is key to security and thus our first security topics will be Cryptography. Internet usage has become part of our daily life, and HTTP is its underlying protocol. Thus, next topic will be study of HTTP protocol and understand various attack mechanisms on web applications.

Network Protocols

HTTP depends upon underlying Transport layer which in turn makes use of Network (IP) layer. Our next security topic will dwell upon Transport Layer security and Network layer security and the role of Firewalls and IDS/IPS systems in understanding attacks and defensive measures. Lastly we will discuss Wireless security, attacks and countermeasures.
