WebSocket Demo

This web page provides a demo of two applications using websockets and shows two useful patterns.
  • server push: server sends data periodically without client asking
  • broadcast: one server message goes to many clients
That makes it a good contrast with ordinary HTTP, where the browser usually has to poll.
  • First one is chat application, which works by opening a chat window.
    When a message is entered in one window, it will appear in chat windows of other browsers.
    This demo shows that a web server can push data to a browser without browser making an explicit request.
  • The other demo simulates stock feed price. Periodically, it generates a number between 160 and 200 and sends as the stock price which is updated
